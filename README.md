# Cryptographie Arabe — TP

Implements César, Affine, and Substitution ciphers on the Arabic alphabet (28 letters),
with frequency analysis attacks to recover the key.

## Usage

Requires Python 3.10+. No external dependencies.
```sh
cd src
python main1.py
python main2.py
```

To compile the report:
```sh
typst compile report/report.typ
```
